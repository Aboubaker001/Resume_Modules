# 📝 ورقة المراجعة - Programmation Linéaire

> مراجعة سريعة قبل الامتحان

---

## 🔄 الشكل القياسي (Forme Standard)

```
Maximiser: Z = c₁x₁ + c₂x₂ + ... + cₙxₙ

Sous contraintes:
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
...
xᵢ ≥ 0 (contraintes de non-négativité)
```

---

## 📌 قبل الفرض - نقاط أساسية

### تحويل القيود

- [ ] **≤ إلى =**: أضف متغير فائض (Slack Variable) `+ s`
- [ ] **≥ إلى =**: اطرح متغير فائض `- s` وأضف متغير اصطناعي `+ a`
- [ ] **= مباشرة**: أضف متغير اصطناعي `+ a`
- [ ] **Min إلى Max**: اضرب دالة الهدف في `-1`

### طريقة السمبلكس

- [ ] **شرط التوقف**: جميع معاملات Z ≥ 0 (للتعظيم)

- [ ] **اختيار Pivot Column**: أكبر معامل سالب في صف Z

- [ ] **اختيار Pivot Row**: أصغر نسبة موجبة
  ```
  Ratio = b / (معامل عمود الـ Pivot)
  (فقط للقيم الموجبة)
  ```

- [ ] **تحديث الجدول**:
  1. قسم صف الـ Pivot على عنصر الـ Pivot
  2. اجعل باقي العمود = 0 بالطرح

### الثنائية (Dualité)

- [ ] **تحويل Primal إلى Dual**:

| Primal (Max) | Dual (Min) |
|:-------------|:-----------|
| n متغيرات | m متغيرات |
| m قيود | n قيود |
| ≤ | متغير ≥ 0 |
| = | متغير حر |
| معاملات الهدف | الطرف الأيمن |
| الطرف الأيمن | معاملات الهدف |
| مصفوفة A | مصفوفة Aᵀ |

- [ ] **نظرية الثنائية القوية**: Z*(Primal) = W*(Dual)

### مسائل النقل

- [ ] **شرط التوازن**: Σ(Offres) = Σ(Demandes)
  - إذا لم يتحقق: أضف مصدر/وجهة وهمية

- [ ] **طريقة الزاوية الشمالية الغربية** (Coin Nord-Ouest):
  1. ابدأ من الزاوية العليا اليسرى
  2. خصص أكبر كمية ممكنة
  3. انتقل يميناً أو للأسفل

- [ ] **طريقة التكلفة الدنيا** (Coût Minimum):
  1. اختر الخلية ذات أقل تكلفة
  2. خصص أكبر كمية ممكنة
  3. كرر حتى تنتهي

- [ ] **عدد المتغيرات الأساسية** = m + n - 1

## 📐 صيغ مهمة

```
الحل الأمثل: عندما جميع Cⱼ - Zⱼ ≤ 0 (للتعظيم)

قيمة Z الأمثل = الطرف الأيمن في صف Z

تكلفة النقل = Σ(كمية × تكلفة الوحدة)
```

| الخطأ | التصحيح |
|:------|:--------|
| نسيان متغير اصطناعي | القيود ≥ و = تحتاج متغير اصطناعي |
