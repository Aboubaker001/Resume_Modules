# 📝 ورقة المراجعة - Économie Numérique et Veille Stratégique

> مراجعة سريعة قبل الامتحان

---

## 📌 قبل الفرض - نقاط أساسية

### الاقتصاد الرقمي (Économie Numérique)

- [ ] **التعريف**: اقتصاد يعتمد على التقنيات الرقمية في الإنتاج والتوزيع والاستهلاك

- [ ] **الخصائص**:
  - اللامادية (Dématérialisation)
  - الفورية (Instantanéité)
  - العالمية (Mondialisation)
  - تأثير الشبكة (Effet de réseau)

- [ ] **التحول الرقمي** (Transformation Digitale):
  - رقمنة العمليات
  - تغيير نماذج الأعمال
  - تحسين تجربة العميل

### التجارة الإلكترونية (E-Commerce)

- [ ] **أنواع التجارة الإلكترونية**:

| النوع | الوصف | مثال |
|:------|:------|:-----|
| B2B | Business to Business | Alibaba |
| B2C | Business to Consumer | Amazon |
| C2C | Consumer to Consumer | eBay, OLX |
| B2G | Business to Government | المناقصات |
| C2B | Consumer to Business | Freelancing |

- [ ] **مكونات التجارة الإلكترونية**:
  - المتجر الإلكتروني (Site web)
  - نظام الدفع (Paiement)
  - اللوجستيك (Livraison)
  - خدمة العملاء (SAV)

- [ ] **وسائل الدفع الإلكتروني**:
  - البطاقات البنكية
  - PayPal
  - العملات الرقمية
  - الدفع عند الاستلام

### اليقظة الاستراتيجية (Veille Stratégique)

- [ ] **التعريف**: عملية منظمة لجمع وتحليل المعلومات حول البيئة الخارجية

- [ ] **أنواع اليقظة**:

| النوع | المحتوى |
|:------|:--------|
| يقظة تنافسية | المنافسون |
| يقظة تكنولوجية | التقنيات الجديدة |
| يقظة تجارية | العملاء والموردون |
| يقظة قانونية | القوانين والتشريعات |

- [ ] **مراحل اليقظة**:
  ```
  1. تحديد الاحتياجات
  2. جمع المعلومات
  3. معالجة وتحليل
  4. نشر المعلومات
  5. اتخاذ القرار
  ```

### الذكاء الاقتصادي (Intelligence Économique)

- [ ] **التعريف**: مجموعة الإجراءات المنسقة للبحث والمعالجة والتوزيع للمعلومات المفيدة للفاعلين الاقتصاديين

- [ ] **الأبعاد الثلاثة**:
  - اليقظة (Veille)
  - حماية المعلومات (Protection)
  - التأثير (Influence)

### أمن المعلومات

- [ ] **التهديدات الرئيسية**:
  - القرصنة (Piratage)
  - التصيد (Phishing)
  - البرمجيات الخبيثة (Malware)
  - سرقة البيانات

- [ ] **مبادئ الأمن** (CIA):
  - السرية (Confidentialité)
  - السلامة (Intégrité)
  - التوفر (Disponibilité)

## 📐 مفاهيم مهمة

### تأثير الشبكة (Effet de Réseau)

```
قيمة الخدمة تزداد مع زيادة عدد المستخدمين
مثال: Facebook, WhatsApp
```

### الاقتصاد التشاركي (Économie Collaborative)

```
مشاركة الموارد بين الأفراد عبر منصات رقمية
مثال: Uber, Airbnb
```

### البيانات الضخمة (Big Data)

```
الخصائص الخمس (5V):
- Volume (الحجم)
- Velocity (السرعة)
- Variety (التنوع)
- Veracity (الصدق)
- Value (القيمة)
```
