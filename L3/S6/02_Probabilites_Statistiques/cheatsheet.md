## 📌 قبل الفرض - نقاط أساسية

### قوانين الاحتمالات الأساسية

- [ ] **احتمال الاتحاد**:
  ```
  P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  ```

- [ ] **احتمال التقاطع** (أحداث مستقلة):
  ```
  P(A ∩ B) = P(A) × P(B)
  ```

- [ ] **الاحتمال الشرطي**:
  ```
  P(A|B) = P(A ∩ B) / P(B)
  ```

- [ ] **نظرية Bayes**:
  ```
  P(A|B) = P(B|A) × P(A) / P(B)
  ```

- [ ] **الاحتمال الكلي**:
  ```
  P(B) = Σ P(B|Aᵢ) × P(Aᵢ)
  ```

### المتغيرات العشوائية

- [ ] **التوقع (Espérance)**:
  ```
  E(X) = Σ xᵢ × P(X = xᵢ)     (متقطع)
  E(X) = ∫ x × f(x) dx        (مستمر)
  ```

- [ ] **التباين (Variance)**:
  ```
  Var(X) = E(X²) - [E(X)]²
  Var(X) = E[(X - μ)²]
  ```

- [ ] **الانحراف المعياري**:
  ```
  σ = √Var(X)
  ```

- [ ] **خصائص**:
  ```
  E(aX + b) = aE(X) + b
  Var(aX + b) = a²Var(X)
  ```

### التوزيعات الاحتمالية

- [ ] **Binomiale** B(n, p):
  ```
  P(X = k) = C(n,k) × pᵏ × (1-p)ⁿ⁻ᵏ
  E(X) = np
  Var(X) = np(1-p)
  ```

- [ ] **Poisson** P(λ):
  ```
  P(X = k) = (λᵏ × e⁻λ) / k!
  E(X) = λ
  Var(X) = λ
  ```

- [ ] **Normale** N(μ, σ²):
  ```
  Z = (X - μ) / σ  (التوحيد)
  
  قاعدة 68-95-99.7:
  P(μ-σ < X < μ+σ) ≈ 68%
  P(μ-2σ < X < μ+2σ) ≈ 95%
  P(μ-3σ < X < μ+3σ) ≈ 99.7%
  ```

### الإحصاء الوصفي

- [ ] **المتوسط**:
  ```
  x̄ = Σxᵢ / n
  ```

- [ ] **الوسيط**: القيمة الوسطى بعد الترتيب

- [ ] **المنوال**: القيمة الأكثر تكراراً

- [ ] **التباين** (العينة):
  ```
  s² = Σ(xᵢ - x̄)² / (n-1)
  ```

### الإحصاء الاستدلالي

- [ ] **فترة الثقة للمتوسط** (σ معلوم):
  ```
  x̄ ± z(α/2) × σ/√n
  ```

- [ ] **اختبار الفرضيات**:
  ```
  H₀: الفرضية الصفرية
  H₁: الفرضية البديلة
  
  إذا |Z| > z(α/2) → نرفض H₀
  ```

## 📐 جداول مرجعية

### قيم Z الشائعة

| مستوى الثقة | α | z(α/2) |
|:-----------:|:-:|:------:|
| 90% | 0.10 | 1.645 |
| 95% | 0.05 | 1.96 |
| 99% | 0.01 | 2.576 |

### التوافقيات

```
C(n,k) = n! / (k! × (n-k)!)
P(n,k) = n! / (n-k)!
```
