# 📚 ملخصات السنة الثالثة ليسانس - إعلام آلي

> مستودع شامل لملخصات ومراجعات مقاييس السنة الثالثة ليسانس في الإعلام الآلي - جامعة الوادي، الجزائر

## 📖 ما هذا المستودع؟

هذا المستودع يحتوي على ملخصات منظمة لجميع مقاييس السنة الثالثة ليسانس (L3) في تخصص الإعلام الآلي. يشمل:

- **ملخصات الفصول** (Résumés) - شروحات مبسطة لكل فصل
- **أوراق المراجعة** (Cheatsheets) - مراجعة سريعة قبل الامتحان
- **قاموس المصطلحات** (Glossaire) - المصطلحات التقنية بالعربية والفرنسية والإنجليزية
- **تمارين محلولة** (TD/TP) - أمثلة تطبيقية
- **امتحانات سابقة** (Examens) - نماذج مع التصحيح

## 📝 كيفية استخدام هذا المستودع

### منهجية الدراسة المقترحة

```
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ ابدأ بقراءة ملخص الفصل في resumes/                      │
│     ↓                                                       │
│  2️⃣ راجع المصطلحات في glossary.md                           │
│     ↓                                                       │
│  3️⃣ حل تمارين TD/TP المتعلقة                                │
│     ↓                                                       │
│  4️⃣ قبل الامتحان: راجع cheatsheet.md                        │
│     ↓                                                       │
│  5️⃣ تدرب على الامتحانات السابقة في exams/                   │
└─────────────────────────────────────────────────────────────┘
```

### نصائح للمراجعة الفعالة

1. **المراجعة اليومية**: خصص 30 دقيقة يومياً لمراجعة ملخص واحد
2. **التكرار المتباعد**: راجع كل فصل بعد يوم، ثم بعد أسبوع، ثم قبل الامتحان
3. **الممارسة النشطة**: لا تكتفِ بالقراءة، حل التمارين وأعد كتابة الملخصات بأسلوبك
